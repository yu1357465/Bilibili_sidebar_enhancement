## 🎨 Bilibili 专栏侧栏美化 (极致沉浸 & 智能响应)

> **一键解锁 B 站专栏的最佳阅读形态。**

> 无论你是新版 Opus 的尝鲜者，还是旧版 Read/CV 的坚守者，本脚本都能为你提供一致、精致且无干扰的阅读体验。

### 🌟 核心亮点：智能响应式布局

本脚本最引以为傲的黑科技——**侧边栏会自动根据你的窗口大小“变换形态”**：

* **💻 大屏模式 (全屏/宽屏)**：侧边栏**自动固定**在文章右侧，紧贴内容，提供最稳固的工具栏体验。

* **📱 窄屏模式 (半屏/小窗)**：侧边栏智能变身为**“侧滑抽屉”**。默认几乎隐藏并保持低透明度，鼠标移入即丝滑弹出，绝不遮挡文字内容。

---

### ✨ 新版专栏 (Opus) 强力增强

**针对新版页面排版痛点进行深度手术：**

1. **🚫 移除多余滚动条**：

**强力干掉底部那个多余、碍眼的横向滚动条**，让页面回归极致整洁，解决由于缩放引起的页面抖动。

2. **📺 铺满模式 (Immersive Mode)**：

新增“铺满”按钮，一键隐藏所有干扰元素，让文章内容完美适配屏幕，视野瞬间开阔。

3. **🔒 状态锁定**：

窄屏下支持“锁定”功能。点击锁图标，抽屉常驻显示；解锁后恢复自动收起，随心所欲。
